# Alan Stines, Ph.D. — Portfolio Website

A professional portfolio website showcasing the academic work, teaching experience, publications, and achievements of Dr. Alan Stines, Associate Professor of Information Technology at Middle Georgia State University.

## Overview

This is a responsive, modern portfolio website built with HTML5, Bootstrap 5, and custom CSS. It provides a comprehensive overview of an academic professional's career, including teaching philosophy, educational background, research publications, and service contributions.

## Features

- **Responsive Design**: Fully mobile-friendly with a collapsible navigation menu
- **Professional Layout**: Clean, modern design built with Bootstrap 5.3.3
- **Multiple Sections**:
  - Hero banner with call-to-action buttons
  - About/Biography section
  - Teaching experience and philosophy
  - Education and credentials
  - Professional experience timeline
  - Service and committee work
  - Publications and research
  - Awards and recognition

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom styling with Bootstrap framework
- **Bootstrap 5.3.3**: Responsive grid system and components
- **JavaScript**: Client-side interactivity
- **CDN**: Bootstrap and fonts from jsDelivr

## How to Use

1. **View Locally**: Simply open `index.html` in a modern web browser
2. **Deploy**: Upload all files to your web hosting service
3. **Customize**: Edit `index.html` to update content and `assets/css/custom.css` to modify styling
